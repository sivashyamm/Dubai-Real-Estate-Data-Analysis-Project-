# 🏙️ Dubai Real Estate Analysis Project

## 📌 Project Overview

This project analyzes Dubai real estate data to understand pricing trends, rental yields, and investment opportunities. The goal is to extract meaningful insights that help investors make better decisions.

---

## 🎯 Objectives

* Analyze property prices across different areas
* Compare rental yields for investment decisions
* Identify high-growth and high-return locations
* Provide business recommendations

---

## 📊 Dataset

The dataset contains information about:

* Property area
* Price per square foot (AED)
* Total property price
* Rental yield (%)
* Property type (Apartment, Villa, etc.)
* Property status (Ready / Off-plan)

---

## 🛠️ Tools & Technologies

* Python
* Pandas
* Matplotlib
* Seaborn

---

## 🔍 Key Analysis Performed

* Data cleaning and preprocessing
* Price analysis by area
* Rental yield analysis
* Year-over-year price trends
* Property type comparison
* Correlation analysis
* Ready vs Off-plan comparison

---

## 📈 Key Insights

* Premium areas like Palm Jumeirah and Downtown Dubai have higher prices
* Areas like JVC and Mirdif offer higher rental yields
* Larger properties have higher total prices
* Expensive areas tend to have lower rental yields
* Off-plan properties are generally cheaper than ready properties

---

## 💡 Business Recommendations

1. **High Rental Income:** Target areas like JVC and Mirdif
2. **Capital Appreciation:** Invest in Palm Jumeirah and Downtown Dubai
3. **Off-plan Investment:** Consider Business Bay for future growth
4. **Short-term Rentals:** Marina and Downtown are ideal
